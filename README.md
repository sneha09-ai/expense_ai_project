# Expense AI Project

An AI-powered Expense Tracker built using Flask, SQLite and Python.

## Features
- Add and manage expenses
- Budget tracking
- Expense analytics
- AI-based financial suggestions
- SQLite database storage

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS

## Project Structure

app.py
database.py
analytics.py
budget.py
advisor.py
ml_model.py
templates/

## Run Locally

1. Clone the repository
2. Install dependencies
3. Run:

python app.py

4. Open:
http://127.0.0.1:5000
